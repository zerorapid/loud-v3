# DISCO | Quick Commerce Enterprise

Elite 10-minute delivery warehouse ecosystem. Built for speed, scale, and industrial-grade reliability.

## 🚀 Ecosystem Features
- **Storefront:** Ultra-sharp, high-performance shopping experience.
- **Command Center:** Real-time logistics, inventory, and financial pulse.
- **Intelligence:** Market-leading analytics and automated supply chain tracking.

## 🛠 Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Database:** Supabase (Postgres)
- **Icons:** Lucide React & Material Symbols

## 📦 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Environment Setup:**
   Create a `.env.local` with your Supabase credentials.

3. **Run Development:**
   ```bash
   npm run dev
   ```

## 🏗 Deployment
Deployed on **Vercel** with continuous synchronization from the `main` branch.
